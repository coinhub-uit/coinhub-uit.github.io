# Web

## Prerequisites

- [Node](https://nodejs.org/en)

## Setup

### Bootstrap

```sh
npm i # or "just b"
```

### Setup `.env`

```env
AUTH_SECRET= # Whatever you want
NEXT_PUBLIC_API_SERVER_URL= # URL to API Server
```

## Dev

- VSCode: <kbd>F5</kbd> to debug _(I may have not set it up)_
- CLI:
  ```sh
  npm run dev # or "just"
  ```
